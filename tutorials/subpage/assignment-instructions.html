Week 2: Multi-Page Website Development Assignment

Building on your Week 1 CRAP analysis, you'll now create a professional multi-page website that showcases your understanding of web design principles and HTML/CSS skills. This assignment transforms your analysis into a complete web portfolio.

Objective

Create a professional four-page website that includes your Week 1 CRAP analysis as one of the pages. You'll learn to organize content across multiple pages, create consistent navigation, and apply professional styling throughout your site.

Required Pages

Your website must include these four pages:

  1. Homepage (index.html) - Professional banner/introduction with navigation
  2. About Page (about.html) - Your background, education, and goals
  3. CRAP Analysis (crap-analysis.html) - Your Week 1 analysis formatted professionally
  4. Contact Page (contact.html) - Your contact information and availability

Technical Requirements

  • Consistent Navigation: Identical navigation menu on all pages
  • Shared CSS: One external stylesheet (style.css) used by all pages
  • Professional Content: Well-written, proofread text throughout
  • Working Links: All internal navigation and external links must function
  • Image Integration: Include your Week 1 CRAP analysis screenshots
  • Semantic HTML: Proper use of headings, paragraphs, lists, and sections

Content Guidelines

Homepage (index.html)

  • Eye-catching banner with your name and professional tagline
  • Brief introduction to who you are and what the site contains
  • Navigation to other pages
  • Professional, welcoming tone

About Page (about.html)

  • Your academic background and current studies
  • Interest in web development and design
  • Professional goals and aspirations
  • Skills you're developing (HTML, CSS, design principles)

CRAP Analysis Page (crap-analysis.html)

  • Your complete Week 1 analysis formatted professionally
  • Include all four screenshots with proper alt text
  • Maintain the 300-400 word count from Week 1
  • Add website URL and date of analysis

Contact Page (contact.html)

  • Your email address (can be school email)
  • Professional social media links (LinkedIn, GitHub)
  • Your location (city/state)
  • Brief note about your availability for opportunities

CSS Styling Requirements

  • Professional color scheme: Choose 2-3 colors that work well together
  • Consistent typography: Use the same fonts throughout
  • Navigation styling: Clear, easy-to-use menu with hover effects
  • Layout structure: Organized content with proper spacing
  • Image styling: Properly sized screenshots with borders/shadows

Submission Instructions

  1. Create all four HTML pages in your GitHub repository
  2. Create a professional CSS stylesheet
  3. Upload your Week 1 CRAP analysis screenshots to the images folder
  4. Test all pages and navigation links
  5. Ensure your site works on GitHub Pages
  6. Submit your GitHub Pages URL in the D2L dropbox

Evaluation Criteria

  • Technical Implementation: Proper HTML structure, working CSS, functional links
  • Content Quality: Professional, well-written content on all pages
  • Design & Consistency: Cohesive visual design across all pages
  • CRAP Analysis Integration: Week 1 analysis properly formatted and included

Important Reminders

  • Use the same navigation menu on every page
  • Link to the same style.css file from all pages
  • Include alt text for all images
  • Test your site in both VS Code Live Preview AND GitHub Pages
  • Make sure all links work before submitting
  • Proofread all content for grammar and spelling
